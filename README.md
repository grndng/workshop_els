# RGB-Kanäle und Pixel-Patch in Binder

Dieses Projekt enthält ein Jupyter-Notebook:

- `rgb_channels_and_patch.ipynb`

Darin kannst du:
- ein Bild per URL laden,
- die RGB-Kanäle nebeneinander ansehen,
- einen Patch (z.B. 10x10 Pixel) mit RGB-Werten in jeder Zelle visualisieren.

## Nutzung in Binder

1. Repository (oder ZIP-Inhalt) nach GitHub hochladen.
2. Mit Binder verbinden.
3. Notebook `rgb_channels_and_patch.ipynb` öffnen.
4. Zellen von oben nach unten ausführen und bei Bedarf `IMAGE_URL` ändern.
